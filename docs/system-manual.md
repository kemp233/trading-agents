# AI Agent Trading System 系统说明书

## 1. 基本信息
- **软件名称**: AI Agent Trading System (aiagentts)
- **版本号**: 1.0.0
- **AppID**: `client_aiagentts_1.0.0`
- **开发主体**: 个人
- **开发人**: 方钰涵
- **适用柜台**: CTP（上期技术综合交易平台）
- **接入期货公司**: 东证期货

## 2. 系统概述
本系统是一个基于 Multi-Agent（多智能体）架构的期货量化交易系统。
- **架构**: 采用事件驱动的五层架构，支持异步协作。
- **下单机制**: 使用 Outbox Pattern（原子下单），确保指令在断网、重启等异常情况下不丢失。
- **支持范围**: 覆盖中国金融期货交易所（CFFEX）、上海期货交易所（SHFE）、大连商品交易所（DCE）、郑州商品交易所（CZCE）、广州期货交易所（GFE）以及上海国际能源交易中心（INE）的所有主力合约。

## 3. 功能说明
### 3.1 交易功能
- **报单**: 支持限价单、市价单。
- **撤单**: 支持单笔撤单、全撤、条件撤单。
- **持仓管理**: 实时同步 CTP 柜台持仓，支持逻辑仓与物理仓对账。

### 3.2 异常监测
- **连接状态**: 实时监测 TradeFront 与 MdFront 连接状态。
- **流量控制**: 统计报单、撤单频率，超过阈值自动熔断。
- **重复报单**: 检测同一秒内针对同一合约的重复逻辑指令。

### 3.3 风险控制
- **六状态机**: 系统具备 Initializing, Ready, Trading, Paused, Emergency, Stopped 六种状态。
- **多级预警**: 达到风控阈值 80% 触发预警，100% 触发自动处置。

### 3.4 应急处置
- **一键全撤**: 在系统进入 Emergency 状态时自动执行。
- **暂停交易**: 禁止新开仓，仅允许平仓。
- **分阶段平仓**: 按比例阶梯式降低风险敞口。

## 4. 技术架构
- **语言**: Python 3.11+
- **并发**: asyncio 异步事件循环
- **持久化**: SQLite (WAL 模式)，保证极高的 IO 性能。
- **接口**: 基于 vn.py 的 `vnpy_ctp` 接口。
- **界面**: Streamlit 仪表盘，提供 00-04 五个核心管理页面。

## 5. 安全说明
- **认证**: 严格遵循东证期货看穿式认证流程（AppID + AuthCode）。
- **隐私**: 所有成交数据、持仓数据本地存储，不上传第三方云端。
- **凭证管理**: 交易密码通过环境变量传递，禁止硬编码在代码或配置文件中。
